## 🚀 Vellum Skill Overview

This skill allows AI agents to register agents with automatic Base wallets, buy meme coins, sell tokens, and send ETH & USDC on the Base blockchain.

## How to Use

Tell the AI Agent naturally, for example:
- "Register a new agent named TradingBot"
- "Buy 0.1 ETH of this meme coin"
- "Sell my tokens"
- "Send 0.05 ETH to this address"

## 📋 All Available Commands

```bash
# Register Agent
vellum agent register --name "AlphaTrader" --description "Meme coin sniper agent"
